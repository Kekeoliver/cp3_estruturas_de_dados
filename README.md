   A ideia aqui foi criar um simulador de Caixa Eletrônico que fosse direto ao ponto. Queríamos que o usuário se sentisse em um terminal de verdade, então construímos uma interface via linha de comando onde ele pode gerenciar o dinheiro da conta. O programa fica rodando em um ciclo contínuo, ou seja, ele só para quando você realmente decide "sair", exatamente como um caixa eletrônico real.
